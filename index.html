Salary Comparison Platform - Input Form

Purpose

The purpose of the "Salary Comparison Project" is to provide benchmarking data about salaries to all participants. With this data each participant can benchmark their own salary position and make an informed decision for their own salary negotiation.

How it works

The collected data is based on voluntary data submission of all participants. To provide data, please fill in the data below and send the material. Each person who has provided data within the last 2 years will be given access to the database, in which one can review the entire submitted data.

Anonymity

The data within the database will be anonymized. Meaning the "name" and the "email" of the data provider will not be visible to the rest of the group. Data marked as "actual" is provided by the user about him-/herself. Data marked "estimated" is based on information about third parties.
